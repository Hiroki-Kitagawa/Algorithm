# Algorithm

# Fizz_Buzz仕様
数字を入力し、1から入力値までを「, 」で結合する。

ただし、3の倍数なら「Fizz」、5の倍数なら「Buzz」、15の倍数なら「FizzBuzz」、いずれでもないならその数を出力する。

以下は入出力例である。

入力：15

出力：1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz
